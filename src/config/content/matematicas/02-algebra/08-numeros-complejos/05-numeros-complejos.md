Un n√∫mero complejo, generalmente denotado con la letra **$z$**, tiene la forma est√°ndar:

### Ejemplo 1: Identificar Partes

Dado el n√∫mero $z = 3 + 5i$, identifica sus partes.

**Razonamiento:**

### Ejemplo 2: Cuidado con los Signos

**Razonamiento:**

### Ejemplo 3: Imaginario Puro

**Razonamiento:**

### Ejemplo 4: Real Puro

Dado $z = 10$, identifica sus partes.

**Razonamiento:**

**Resultado:**

### Ejemplo 5: Igualdad de Complejos

Si $3 + 4i = x + yi$, ¬øcu√°nto valen $x$ y $y$?

**Razonamiento:**
Para que dos complejos sean iguales, sus partes deben coincidir exactamente:
- Real con Real: $3 = x$
- Imaginaria con Imaginaria: $4 = y$

**Resultado:**

$$

$$
