# üìê Cuadrado de un Trinomio

En esta lecci√≥n aprenderemos a calcular el cuadrado de un trinomio, extendiendo el concepto del cuadrado de un binomio.

---

## üìñ F√≥rmula del cuadrado de un trinomio

El cuadrado de un trinomio $(a + b + c)^2$ sigue la f√≥rmula:

$$
(a + b + c)^2 = a^2 + b^2 + c^2 + 2ab + 2ac + 2bc
$$

**En palabras:** El cuadrado de un trinomio es igual a la suma de los cuadrados de cada t√©rmino, m√°s el doble de cada uno de los productos de los t√©rminos tomados de dos en dos.

---

## üìñ Demostraci√≥n

Podemos demostrar esta f√≥rmula agrupando t√©rminos:

$$
(a + b + c)^2 = [(a + b) + c]^2
$$

Aplicamos el cuadrado de un binomio:

$$
= (a + b)^2 + 2(a + b)(c) + c^2
$$

$$
= a^2 + 2ab + b^2 + 2ac + 2bc + c^2
$$

Reagrupando:

$$
= a^2 + b^2 + c^2 + 2ab + 2ac + 2bc
$$

---

## üìñ Regla pr√°ctica

Para calcular $(a + b + c)^2$:

1. **Cuadrados:** Eleva al cuadrado cada t√©rmino ‚Üí $a^2$, $b^2$, $c^2$
2. **Dobles productos:** Calcula el doble de cada producto entre pares de t√©rminos ‚Üí $2ab$, $2ac$, $2bc$
3. **Suma todo**

### Diagrama visual

Los productos a calcular son:

| | $a$ | $b$ | $c$ |
|:--:|:--:|:--:|:--:|
| $a$ | $a^2$ | $2ab$ | $2ac$ |
| $b$ | ‚Äî | $b^2$ | $2bc$ |
| $c$ | ‚Äî | ‚Äî | $c^2$ |

---

## üìñ Ejemplos con suma de t√©rminos

### Ejemplo 1

Desarrolla: $(x + y + z)^2$

**Cuadrados:** $x^2$, $y^2$, $z^2$

**Dobles productos:** $2xy$, $2xz$, $2yz$

$$
(x + y + z)^2 = x^2 + y^2 + z^2 + 2xy + 2xz + 2yz
$$

$$
\boxed{x^2 + y^2 + z^2 + 2xy + 2xz + 2yz}
$$

### Ejemplo 2

Desarrolla: $(x + 2y + 3)^2$

Identificamos: $a = x$, $b = 2y$, $c = 3$

**Cuadrados:**
- $x^2$
- $(2y)^2 = 4y^2$
- $3^2 = 9$

**Dobles productos:**
- $2(x)(2y) = 4xy$
- $2(x)(3) = 6x$
- $2(2y)(3) = 12y$

$$
(x + 2y + 3)^2 = x^2 + 4y^2 + 9 + 4xy + 6x + 12y
$$

$$
\boxed{x^2 + 4y^2 + 4xy + 6x + 12y + 9}
$$

### Ejemplo 3

Desarrolla: $(2a + 3b + c)^2$

**Cuadrados:**
- $(2a)^2 = 4a^2$
- $(3b)^2 = 9b^2$
- $c^2$

**Dobles productos:**
- $2(2a)(3b) = 12ab$
- $2(2a)(c) = 4ac$
- $2(3b)(c) = 6bc$

$$
(2a + 3b + c)^2 = 4a^2 + 9b^2 + c^2 + 12ab + 4ac + 6bc
$$

$$
\boxed{4a^2 + 9b^2 + c^2 + 12ab + 4ac + 6bc}
$$

## üìñ Ejemplos con signos negativos

Cuando alg√∫n t√©rmino es negativo, debemos tener cuidado con los signos en los dobles productos.

### Ejemplo 4

Desarrolla: $(x - y + z)^2$

Identificamos: $a = x$, $b = -y$, $c = z$

**Cuadrados:**
- $x^2$
- $(-y)^2 = y^2$
- $z^2$

**Dobles productos:**
- $2(x)(-y) = -2xy$
- $2(x)(z) = 2xz$
- $2(-y)(z) = -2yz$

$$
(x - y + z)^2 = x^2 + y^2 + z^2 - 2xy + 2xz - 2yz
$$

$$
\boxed{x^2 + y^2 + z^2 - 2xy + 2xz - 2yz}
$$

### Ejemplo 5

Desarrolla: $(a - b - c)^2$

Identificamos: $a = a$, $b = -b$, $c = -c$

**Cuadrados:**
- $a^2$
- $(-b)^2 = b^2$
- $(-c)^2 = c^2$

**Dobles productos:**
- $2(a)(-b) = -2ab$
- $2(a)(-c) = -2ac$
- $2(-b)(-c) = 2bc$

$$
(a - b - c)^2 = a^2 + b^2 + c^2 - 2ab - 2ac + 2bc
$$

$$
\boxed{a^2 + b^2 + c^2 - 2ab - 2ac + 2bc}
$$

### Ejemplo 6

Desarrolla: $(2x - 3y - 1)^2$

**Cuadrados:**
- $(2x)^2 = 4x^2$
- $(-3y)^2 = 9y^2$
- $(-1)^2 = 1$

**Dobles productos:**
- $2(2x)(-3y) = -12xy$
- $2(2x)(-1) = -4x$
- $2(-3y)(-1) = 6y$

$$
(2x - 3y - 1)^2 = 4x^2 + 9y^2 + 1 - 12xy - 4x + 6y
$$

$$
\boxed{4x^2 + 9y^2 - 12xy - 4x + 6y + 1}
$$

### Ejemplo 7

Desarrolla: $(x + y - 2z)^2$

**Cuadrados:**
- $x^2$
- $y^2$
- $(-2z)^2 = 4z^2$

**Dobles productos:**
- $2(x)(y) = 2xy$
- $2(x)(-2z) = -4xz$
- $2(y)(-2z) = -4yz$

$$
(x + y - 2z)^2 = x^2 + y^2 + 4z^2 + 2xy - 4xz - 4yz
$$

$$
\boxed{x^2 + y^2 + 4z^2 + 2xy - 4xz - 4yz}
$$

## üìñ Casos especiales

### Ejemplo 8: Con fracciones

Desarrolla: $\left(x + \frac{1}{2} + y\right)^2$

**Cuadrados:**
- $x^2$
- $\left(\frac{1}{2}\right)^2 = \frac{1}{4}$
- $y^2$

**Dobles productos:**
- $2(x)\left(\frac{1}{2}\right) = x$
- $2(x)(y) = 2xy$
- $2\left(\frac{1}{2}\right)(y) = y$

$$
\left(x + \frac{1}{2} + y\right)^2 = x^2 + y^2 + \frac{1}{4} + x + 2xy + y
$$

$$
\boxed{x^2 + y^2 + 2xy + x + y + \frac{1}{4}}
$$

## üìñ Generalizaci√≥n: Cuadrado de un polinomio

La f√≥rmula se puede extender a cualquier n√∫mero de t√©rminos:

$$(a_1 + a_2 + ... + a_n)^2 = \sum_{i=1}^{n} a_i^2 + 2\sum_{i<j} a_i a_j$$

Es decir:
- La suma de los cuadrados de todos los t√©rminos
- M√°s el doble de todos los productos posibles entre pares diferentes
