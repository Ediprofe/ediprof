## üìê Las F√≥rmulas

**Suma de cubos:**

$$
\boxed{a^3 + b^3 = (a + b)(a^2 - ab + b^2)}

\boxed{a^3 - b^3 = (a - b)(a^2 + ab + b^2)}

**C√≥mo recordarlas:**
- El primer par√©ntesis tiene el **mismo signo** que la expresi√≥n original.

### Ejemplo 1: Suma de cubos b√°sica

Factoriza: $x^3 + 8$

**Datos:**
- $x^3 = (x)^3$
- $8 = (2)^3$

**Razonamiento:**

1. Identificamos $a = x$ y $b = 2$.

2. Aplicamos la f√≥rmula de suma.

3. Primer factor:

4. Segundo factor:

**Resultado:** $\boxed{(x + 2)(x^2 - 2x + 4)}$

### Ejemplo 2: Diferencia de cubos b√°sica

**Datos:**

**Razonamiento:**

### Ejemplo 3: Suma con coeficientes

**Datos:**

**Razonamiento:**

**Resultado:** $\boxed{(3a - 4b)(9a^2 + 12ab + 16b^2)}$

### Ejemplo 5: Con factor com√∫n
